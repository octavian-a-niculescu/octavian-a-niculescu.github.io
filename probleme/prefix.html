La ora de Informatică, doamna profesoară dorește să îi grupeze pe elevi in echipe dar cu condiția ca numele fiecărei echipe să reprezinte cel mai mare prefix comun al numelor tuturor elevilor care o reprezintă.Astfel, provocarea este ca aceștia să implementeze o metodă ce folosește Divide et Impera pentru calculul celui mai mare prefix comun (C.M.M.P.C).

Date de intrare: numarul de stringuri n, un sir de n stringuri
Date de iesire: Un string sau un caracter reprezentand prefixul maxim

//O functie de tip string ce returneaza prefixul comun a doua string-uri
string prefixBetweenTwoStrings(string str1, string str2)
{
string result;
int n1 = str1.length(), n2 = str2.length();
 
for (int i = 0, j = 0; i <= n1 - 1 && j <= n2 - 1; i++, j++)
{
if (str1[i] != str2[j])
break;
result.push_back(str1[i]);
}
return (result);
}
 
/*Functia recursiva ce imparte input-ul in jumatati pana ce ajunge la o solutie partiala formata din cel mult 2 elemente
* strLeft-jumatatea din stanga a sirului de string-uri
* strRight-jumatatea din dreapta a sirului de string-uri
*/
string LongestCommonPrefix(vector str, int start, int finish)
{
if (start == finish)
return (str[start]);//sirul contine un singur element care este el insusi prefixul comun
 
if (start < finish)
{
int m = (start + finish) / 2;//injumatatirea input-ului
                        
string strLeft = LongestCommonPrefix(str, start, m);
                       //autoapel al sirului stang si drept pana cand acestea vor avea rezultate partiale
string strRight = LongestCommonPrefix(str, m + 1, finish);

return (prefixBetweenTwoStrings(strLeft, strRight));
}
}
 
void readFromUser()
{
int n;
cout << "Numarul de string-uri: ";
cin >> n;
int nr = 1;
string word;
vector str;
while (getline(cin, word) && nr <= n)//functia getline citeste toate caracterele din "word" pana la tasta ENTER
{
if (word[0] >= 'a' && word[0] <= 'z' || word[0] >= 'A' && word[0] <= 'Z')//conditia ignora caracterele care nu sunt litere din input
{
str.push_back(word);
nr++;
}
if (nr > n)
break;
}
string answer = LongestCommonPrefix(str, 0, n - 1);
if (answer.size())
cout << "Cel mai lung prefix si numele echipei este: " << answer;
else
cout << "Nu exista un prefix comun!";
}
Explicatie rezolvare: • Algoritmul este asemanator algoritmului pentru sortare Merge Sort
• La fiecare pas sirul este impartit in siruri de lungime mai mica pana cand ajunge la o solutie partiala formata din unul sau doua string-uri
• Odata ajuns la doua siruri de lungime 1,algoritmul recursiv apeleaza functia iterativa prefixBetweenTwoStrings ce returneaza prefixul comun dintre doua string-uri
Complexitate: O(m*n)
