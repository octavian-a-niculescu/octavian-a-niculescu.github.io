Aflat în New York printre zgârie-nori, Marius dorește să se urce într-o clădire care respectă următoarea condiție: cei patru vecini ai săi (stânga, dreapta, sus, jos) să fie mai scunzi sau la fel de înalți. Pentru a-și simplifica decizia, Marius dorește să noteze în formă de matrice zgârie-norii din New York. Ce înălțime va avea clădirea în care va urca Marius?
Observatii:
  • Cladirile aflate pe diagonală nu sunt considerate ca fiind vecini
  • Cladirea in care se va urca Marius nu e neapărat cea mai inaltă
  • Cladirea se poate afla și la marginea sau colțul matricei
#include "iostream"
void citire(int& linie, int& coloana, int matrice[][100])
{
std::cout << "Introduceti numarul de linii: ";
std::cin >> linie;
std::cout << "Introduceti numarul de coloane: ";
std::cin >> coloana;
std::cout << "Introduceti matricea: ";
for (int index1 = 0; index1 < linie; index1++)
for (int index2 = 0; index2 < coloana; index2++)
std::cin >> matrice[index1][index2];
}
int cautaMijlMax(int matrice[][100], int coloana, int mijloc, int& max)
{
int indexMax = 0;
for (int index = 0; index < coloana; index++)
{
if (max < matrice[mijloc][index])
{
max = matrice[mijloc][index];
indexMax = index;
}
}
                         
return indexMax;
}
                         
int gasesteElementulVarf(int matrice[][100], int linie, int coloana, int mijloc)
{
int mijlocMax = 0;
int indexMijlMax = cautaMijlMax(matrice, coloana, mijloc, mijlocMax);
if (mijloc == 0 or mijloc == linie - 1)
return mijlocMax;
if (mijlocMax >= matrice[mijloc - 1][indexMijlMax] and mijlocMax >= matrice[mijloc + 1][indexMijlMax])
return mijlocMax;
if (mijlocMax < matrice[mijloc - 1][indexMijlMax])
return gasesteElementulVarf(matrice, linie, coloana, mijloc - mijloc / 2);
return gasesteElementulVarf(matrice, linie, coloana, mijloc + mijloc / 2);
}
int main()
{
int matrice[100][100], linie, coloana;
citire(linie, coloana, matrice);
std::cout << "Posibila clădire în care Marius se va urca are înalțimea: " << gasesteElementulVarf(matrice, linie, coloana, linie / 2)<<" de metri.";
return 0;
}
Explicatie rezolvare: În cadrul funcției main se declară variabilele întregi linie și coloană alături de tabloul bidimensional matrice[100][100]. Se apelează funcția „citire” care se ocupă de citirea datelor prin referință. După aceea se apelează funcția „gaseșteElementulVârf” care primește patru parametrii: matrice, linie, coloana și linie/2 (în funcție aceasta se va numi mijloc) care reprezintă mijlocul matricei din punctul de vedere al liniilor. În interiorul subprogramului se mai apelează funcția „cautaMijlMax” care determină valoarea cea mai mare de pe linia de mijloc(maximul) a matricei, cât și numărul coloanei (poziția) pe care se află. Revenind în cadrul funcției „gaseșteElementulVârf” se verifică dacă mijlocul aparține primei sau ultimei linii, și dacă într-adevăr aparține se returnează valoarea maximă. În cazul în care nu aparține primei condiții, se verifică dacă este cel mai mare dintre vecinii de sus și de jos. În cel de-al treilea if se verifică dacă mijlocul este mai mic decât vecinul său de deasupra, atunci se reapelează funcția „gaseșteElementulVârf”, însă cel de al patrulea parametru va primi valoarea de mijloc-mijloc/2, întrucât există deasupra elementului curent o valoare mai mare. Dacă niciuna din condiții nu este respectată înseamnă că a rămas cazul în care vecinul de jos este mai mare, iar funcția „gaseșteElementulVârf” este reapelată ca în condiția adineauri prezentată, însă de data aceasta cel de al patrulea parametru va primi valoarea mijloc+mijloc/2. La final se returnează valoarea și se afișează pe ecran înălțimea clădirii în care Marius se va urca.
Complexitate: T(linie, coloana)=O(coloana* log(linie))
Demonstratie complexitate:
Fiind matrice de 4 linii, se împarte la 2 (ca la căutarea binară) pentru a ușura găsirea unui element vârf, astfel submatricea formată conține primele două rânduri. Parametrul mijloc va avea așadar valoarea 1 (dacă indexarea începe de la 0) sau 2 (dacă indexarea începe de la 1). Se caută pe cea de-a doua linie elementul maxim care se află pe coloana 2 sau 3 și se returnează numărul coloanei pe care se află și totodată se trimite elementul prin referință. Fiind un algoritm care înjumătățește numărul de linii, dar care totuși parcurge fiecare coloană, algoritmul scris are complexitatea de mai sus.
Pasi executie:
  1. Se împarte matricea în jumătăți în funcție de linii =>
  2. Se selectează jumătatea superioară =>
  3. Se caută pe fiecare coloană de pe linia doi un posibil element vârf =>
  4. Se returnează 18 care este elementul vârf și se oprește execuția programului
